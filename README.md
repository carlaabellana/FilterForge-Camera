# FilterForge-Camera
![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)
![React Native](https://img.shields.io/badge/React_Native-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)

A small two-screen mobile app built with React Native to practice UI state, form validation, tab navigation, and real-time camera overlays.

The app includes:
- **Profile** screen: a form with live preview
- **Camera with Filters** screen: front camera preview with selectable overlay accessories
